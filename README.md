🏛️ **Village Temple Profile and Pre-Marriage Registration System**
A comprehensive web-based system designed to manage village profiles, temple records, and pre-marriage registrations digitally.
The system helps temple administrators and villagers handle registrations, payments, and certificate generation efficiently.

🚀 **Features**

🔐 **User Authentication**
Secure sign-up and sign-in system for devotees.
User account management and authentication.
💍 **Marriage Registration**
Multi-step registration form for collecting detailed marriage information
Automatic Application Number generation.
🧑‍💼 **Admin Panel**
Admin dashboard to view and manage registration records.
Synchronization of application numbers and records.
🏡 **Village Profile**
Integration with village administration details.
Manage village and temple related information.
💳 **Payment Integration**
Integrated Razorpay payment gateway for registration fees.
📜 **Certificate Generation**
Automatically generate digital marriage certificates after successful registration.

🛠️ **Technical Stack**
**Backend:** Java 17,Spring Boot,Spring Data JPA,Maven
**Frontend:** HTML5,CSS3,JavaScript (ES6+),FontAwesome
**Database:** MySQL
**Payment Gateway:** Razorpay

⚙️ **Deployment**
To run this project locally:
        1️⃣ Clone the repository
        2️⃣ Configure MySQL database
        3️⃣ Update application.properties in Spring Boot
        4️⃣ Run the backend using Maven
        5️⃣ Open the frontend in the browser
Refer to the local setup instructions for manual execution.
